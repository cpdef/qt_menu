# qt_menu
a qt menu for sway wm

Keys: 
* Escape to quit
* Return to run the current typed command

Click on an symbol to launch the wanted application

------------
# Build instructions
Dependencies: CMake, Qt5

    mkdir build && cd build
    cmake .. 
    make
    ./qt_menu
    # start typing
